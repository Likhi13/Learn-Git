pull n fetch diff:
+---------+     +---------+     +-------------+     +--------------+
|  Work   | --> |  Stage  | --> |  Local Repo | <-- |  Remote Repo |
+---------+     +---------+     +-------------+     +--------------+
       ^                ^                ^                 ^
       |                |                |                 |
       |                |                |                 |
       |                |                |   git fetch     |
       |                |                |<----------------|
       |                |                |   (get info but don’t put in work)
       |                |                |
       |                |                |   git pull
       |                |<---------------|
       |                |   (get info & add it in work)
       |
    git pull = git fetch + git merge
    git pull origin main  → changes merged to main from origin
